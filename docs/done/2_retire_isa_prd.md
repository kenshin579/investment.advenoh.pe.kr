## 블로그 목차 구성

### 1. 개요

- ISA는 단순한 비과세 계좌가 아니라, **연금계좌와 연계하면 절세 효과가 극대화**되는 계좌다
- 이 글은 ISA의 기본 구조부터 만기 후 연금 전환 전략까지 정리하는 스터디 노트
- 이전 글([연금저축 납입·인출 절세 전략](/etc/pension-savings-withdrawal-strategy/))에서 간단히 언급한 ISA 연계 전략을 상세히 다룬다

**이 글을 읽으면 알 수 있는 것:**
- ISA 계좌의 종류와 비과세 혜택
- ISA 만기 후 연금계좌로 전환하는 방법과 추가 세액공제
- ISA → 연금저축 전환 시 비과세 재원이 되는 구조
- 전환 시 주의사항 (60일 기한 등)

---

### 2. ISA란?

#### 2.1 ISA의 정의

- 개인종합자산관리계좌 (Individual Savings Account)
- 하나의 계좌에서 예금, 펀드, ETF 등 다양한 금융상품에 투자할 수 있는 만능 계좌
- 핵심 혜택: **손익 통산 후 비과세 + 초과분 분리과세 9.9%**

#### 2.2 ISA 종류

| 구분 | 일반형 | 서민형 | 농어민형 |
|------|-------|-------|---------|
| 가입 조건 | 만 19세 이상 | 총급여 5,000만 원 이하 | 농어민 |
| 비과세 한도 | 200만 원 | 400만 원 | 400만 원 |
| 초과분 세율 | 9.9% 분리과세 | 9.9% 분리과세 | 9.9% 분리과세 |

#### 2.3 ISA 기본 스펙

| 항목 | 내용 |
|------|------|
| 의무 가입 기간 | 3년 |
| 연간 납입 한도 | 2,000만 원 (미사용분 이월 가능, 총 1억 원) |
| 투자 가능 상품 | 예금, 펀드, ETF, 리츠 등 |
| 과세 방식 | 손익 통산 후 비과세 한도 적용 → 초과분 9.9% 분리과세 |

#### 2.4 손익 통산이란?

- 일반 계좌: A펀드 +300만 원, B펀드 -200만 원 → A펀드 수익 300만 원에 과세
- ISA 계좌: A펀드 +300만 원, B펀드 -200만 원 → 순이익 100만 원에만 과세 (비과세 한도 이내면 세금 0원)
- 이 손익 통산이 ISA의 가장 큰 장점

---

### 3. ISA 만기 후 연금계좌 전환 전략 (핵심)

#### 3.1 전환 시 추가 세액공제

- ISA 만기 해지 후 **60일 이내**에 연금계좌(연금저축 또는 IRP)로 이전
- 이전 금액의 10%, 최대 **300만 원** 추가 세액공제
- 기존 연금저축+IRP 세액공제 900만 원 + ISA 전환 300만 원 = **총 1,200만 원** 세액공제 가능

#### 3.2 세액공제 시뮬레이션

| 항목 | 금액 |
|------|------|
| 연금저축 납입 | 600만 원 (세액공제) |
| IRP 납입 | 300만 원 (세액공제) |
| ISA 전환 추가 공제 | 300만 원 (세액공제) |
| **합계** | **1,200만 원** |
| 환급액 (13.2%) | **158.4만 원** |
| 환급액 (16.5%) | **198만 원** |

#### 3.3 전환 금액과 추가 공제 관계

| ISA 전환 금액 | 추가 세액공제 대상 (10%) | 실제 추가 공제 |
|-------------|----------------------|--------------|
| 1,000만 원 | 100만 원 | 100만 원 |
| 2,000만 원 | 200만 원 | 200만 원 |
| 3,000만 원 이상 | 300만 원 (한도) | 300만 원 |

#### 3.4 전환 후 비과세 재원 확보

- ISA에서 연금계좌로 이전한 금액 중 **추가 세액공제를 받지 않은 부분은 비과세 재원**이 된다
- 예: 3,000만 원 이전 → 300만 원 세액공제 대상 + **2,700만 원 비과세 재원**
- 이 비과세 재원은 연금 수령 시 세금 없이 인출 가능 (연금저축 인출 순서 1순위)
- 연금저축 글의 "인출 순서" 전략과 직접 연결되는 부분

---

### 4. ISA 활용 타임라인

#### 4.1 가입부터 전환까지 로드맵

```
1년차~3년차: ISA에 매년 2,000만 원 납입 + ETF/펀드 투자
          (동시에 연금저축 600만 원 + IRP 300만 원 별도 납입)
3년차 만기: ISA 해지 → 60일 이내 연금계좌로 이전
          → 추가 세액공제 300만 원 확보
          → 나머지는 비과세 재원으로 확보
만기 후: 새 ISA 재가입 → 다시 3년 운용 → 반복
```

#### 4.2 ISA 만기 시 선택지

| 선택지 | 설명 | 추천 |
|-------|------|------|
| 연금계좌 전환 | 추가 세액공제 + 비과세 재원 확보 | 장기 투자자에게 추천 |
| 재가입 (재설정) | ISA 비과세 혜택 리셋 | 단기 자금 필요 시 |
| 일반 해지 | 비과세/분리과세 혜택 후 현금화 | 급전 필요 시 |

---

### 5. 주의사항

- **60일 기한**: 만기 해지 후 60일 이내에 연금계좌로 이전해야 추가 세액공제 혜택 적용
- **납입 한도 별도**: ISA 전환 금액은 연금계좌의 연간 납입 한도(1,800만 원)와 별도로 인정
- **전환 후 인출 제한**: 연금계좌로 이전하면 연금계좌의 인출 규칙을 따른다 (만 55세 이후 연금 수령)
- **세액공제 받은 300만 원**: 연금 수령 시 연금소득세(3.3~5.5%) 부과
- **세액공제 안 받은 나머지**: 비과세 재원으로 세금 없이 인출 가능

---

### 6. 정리 및 체크리스트

- ISA 3년 만기 도래 시 연금계좌 전환 검토
- 60일 이내 전환 필수 (기한 놓치면 추가 세액공제 불가)
- 최소 3,000만 원 이상 전환해야 추가 세액공제 300만 원 한도 달성
- 전환 후 비과세 재원은 인출 순서 1순위로 활용
- 전환 완료 후 새 ISA 재가입하여 사이클 반복
