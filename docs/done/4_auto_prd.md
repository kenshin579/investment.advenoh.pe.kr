# 미래에셋증권 서버자동주문 가이드 — 블로그 포스트 작성 PRD

## 1. 개요

### 1.1 목적

미래에셋증권의 서버자동주문 기능을 소개하고, MTS(모바일)와 HTS(PC)에서 Stoploss 및 Trailing Stop을 설정하는 방법을 단계별로 정리한다. 국내/해외 주식 모두에 대한 활용법을 다루며, Stoploss(손절매 자동화)가 이 글의 핵심이다.

### 1.2 배경

- 직장인 등 장 시간에 실시간으로 모니터링하기 어려운 투자자가 많다
- 서버자동주문은 증권사 서버에서 조건을 감시하므로 PC/스마트폰이 꺼져도 동작한다는 점이 일반 예약주문과의 핵심 차이
- MTS에서는 간편 설정, HTS에서는 Trailing Stop 등 고급 전략까지 가능
- Notion에서 스터디 및 실제 테스트 완료 후 초안 작성 단계

### 1.3 콘텐츠 성격

- 차트 분석 시리즈가 아닌, **증권사 기능 활용 가이드** 성격의 실용적 콘텐츠
- 시리즈 지정 없음 (독립 글)
- 스크린샷 위주로 따라하기 쉬운 구성

## 4. 콘텐츠 목차 구조 (안)

```markdown
# 1. 개요
- 서버자동주문이란 무엇인가
- 왜 사용해야 하는가 (일반 예약주문과의 차이)
- 서버자동주문의 주요 특징 정리
  - 증권사 서버에서 실시간 감시 (PC/스마트폰 꺼도 동작)
  - 최대 90일까지 조건 유지 (해외는 약 30일)
  - 매도/Stoploss, 목표가, Trailing Stop 등 다양한 조건 설정 가능

# 2. 핵심 개념 이해하기
## 2.1 Stoploss(손절매)란?
- 주가가 설정한 손실률 이하로 떨어지면 자동 매도
- 추가 하락에 대한 손실을 방지하는 기법
## 2.2 Trailing Stop(트레일링 스탑)이란?
- 주가 상승 시 고점을 자동 추적하며 매도 기준을 올려가는 전략
- 고점 대비 일정 폭(비율/금액) 하락 시 자동 매도
- Stoploss와의 차이: 손절 vs 이익 극대화

# 3. MTS에서 서버자동주문 설정하기
## 3.1 자동매도 설정 방법
- MY자산 > 보유 종목 선택 > 자동 버튼 클릭
- 감시조건, 주문조건, 감시 유효시간 설정
- 저장후 실행 > 확인
## 3.2 자동주문 현황 확인 및 관리
- 등록된 자동주문 확인/수정 방법

# 4. HTS에서 서버자동주문 설정하기
## 4.1 서버자동주문 화면 접근
- 주식특화주문 > 서버자동주문 메뉴
- 기본 화면 구성 설명
## 4.2 자동매도 조건 설정
- 계좌/종목/수량/감시조건 설정
- Stoploss 조건 설정
- 목표가 조건 설정
## 4.3 Trailing Stop 설정
- 이익실현 기준가 설정
- 트레일 기준(비율/가격) 설정
- 동작 원리: 기준가 돌파 → 고점 추적 → 일정 폭 하락 시 매도
## 4.4 감시 관리
- 감시 시작/중단/수정 방법
- 이미 실행된 조건은 해제 불가

# 5. 해외주식 서버자동주문
## 5.1 국내 주식과의 공통점과 차이점
- 설정 방법은 국내 주식과 거의 동일
- 감시 기간: 약 30일 (국내 90일보다 짧음)
- 원화/외화 주문 모두 가능
## 5.2 MTS/HTS에서 해외주식 자동주문 설정
- 해외주식 서버자동주문 화면 접근 방법
## 5.3 해외주식 자동주문 시 유의사항
- 거래소별 거래 시간
- 환율 변동 영향

# 6. 서버자동주문 활용 팁과 주의사항
- 신규잔고편입: 신규 매수 종목에 일괄 조건 적용
- 서버자동주문 vs HTS 스탑로스의 차이
  - 서버자동주문: 서버 감시, PC 꺼도 동작
  - HTS 스탑로스: HTS 활성화 시에만 동작 (PC 켜면 바로 실행)
- 익절/손절 조건 복수 동시 설정 가능

# 7. 마무리
```
