# 이동평균선 완벽 가이드 — 블로그 포스트 작성 PRD

## 1. 개요

### 1.1 목적

이동평균선(Moving Average)의 개념과 원리를 체계적으로 정리하고, 실제 투자에서 어떻게 활용할 수 있는지를 데이터와 차트 중심으로 설명한다. 기술적 분석의 가장 기본이 되는 이동평균선을 이해하여, 매수/매도 시점을 판단하는 데 활용할 수 있도록 돕는다.

### 1.2 배경

- 이동평균선은 기술적 분석에서 가장 널리 사용되는 지표로, 초보 투자자가 반드시 이해해야 할 개념
- 골든크로스/데드크로스 같은 용어는 자주 접하지만, 단순 신호가 아닌 맥락에 따른 해석이 중요
- 기존 Notion 초안이 있으나 미완성 상태 — 내용을 보강하고 구조화하여 완성도 높은 블로그 글로 재작성

### 1.3 기존 초안 분석

## 4. 콘텐츠 목차 구조

### 4.1 최종 목차 (안)

```markdown
# 1. 개요
- 이동평균선이란 무엇인가
- 왜 이동평균선을 알아야 하는가
- 이 글에서 다루는 범위

# 2. 이동평균선의 기본 개념
## 2.1 이동평균선(Moving Average)의 정의
- 일정 기간 동안의 종가를 평균한 값을 연결한 선
- 그랜빌(J.E. Granville)이 고안
- 계산 예시: 5일 이평선 = 최근 5일 종가 합 / 5
## 2.2 단기/중기/장기 이동평균선
- 단기: 5일, 10일, 20일(세력선)
- 중기: 60일(수급선), 120일(경기선)
- 장기: 240일(대세이평선)
- 각 이평선이 의미하는 것
## 2.3 이동평균선의 계산 방법
- 단순이동평균(SMA): 모든 데이터에 동일 가중치를 부여하여 산출
- 계산 예시와 함께 직관적으로 이해
(차트 예시)

# 3. 이동평균선의 배열 — 정배열과 역배열
## 3.1 정배열이란
- 주가 > 단기 > 중기 > 장기 순서
- 상승 추세 지속 신호
- 매도 시점은 아직 오지 않았다
## 3.2 역배열이란
- 장기 > 중기 > 단기 > 주가 순서
- 하락 추세 지속 신호
- 골든크로스 기미가 보일 때까지 관망
## 3.3 배열 전환의 의미
- 정배열 → 역배열: 추세 전환 경고
- 역배열 → 정배열: 새로운 상승 추세 시작 가능성
(Mermaid 다이어그램: 배열 상태 전환 흐름)

# 4. 골든크로스와 데드크로스
## 4.1 골든크로스(Golden Cross)
- 단기 이평선이 장기 이평선을 아래에서 위로 돌파
- 매수 신호로 해석
- 주요 조합: 20일-60일, 50일-200일
## 4.2 데드크로스(Dead Cross)
- 단기 이평선이 장기 이평선을 위에서 아래로 돌파
- 매도 신호로 해석
## 4.3 크로스 신호의 한계
- 후행 지표의 특성 — 이미 추세가 진행된 후 나타남
- 횡보장에서 빈번한 속임 신호(Whipsaw) 발생
- 단독 사용보다 다른 지표와 함께 활용해야 하는 이유
(차트 예시: 골든크로스/데드크로스)

# 5. 이동평균선의 지지와 저항
## 5.1 이평선이 지지선 역할을 하는 경우
- 상승 추세에서 주가가 이평선 근처까지 하락 후 반등
- 20일선, 60일선, 120일선 지지 사례
## 5.2 이평선이 저항선 역할을 하는 경우
- 하락 추세에서 주가가 이평선 근처까지 상승 후 재하락
## 5.3 지지/저항 돌파의 의미
- 주요 이평선 이탈 시 추세 전환 가능성
(차트 예시)

# 6. 그랜빌의 이동평균선 8법칙
## 6.1 매수 신호 4가지
- 1법칙: 이평선 하락 후 횡보/상승 전환 시 주가가 이평선 상향 돌파
- 2법칙: 이평선 상승 중 주가가 이평선 아래로 하락했다가 재돌파
- 3법칙: 주가가 이평선 위에서 하락하다 이평선에 닿지 않고 재상승
- 4법칙: 주가가 이평선을 크게 하회 — 기술적 반등 기대 (이격도 개념은 별도 글에서 상세히 다룰 예정)
## 6.2 매도 신호 4가지
- 5법칙: 이평선 상승 후 횡보/하락 전환 시 주가가 이평선 하향 돌파
- 6법칙: 이평선 하락 중 주가가 이평선 위로 상승했다가 재하락
- 7법칙: 주가가 이평선 아래에서 상승하다 이평선에 닿지 못하고 재하락
- 8법칙: 주가가 이평선을 크게 상회 — 차익 실현 고려
(Mermaid 다이어그램 또는 차트 예시: 8법칙 시각화)

# 7. 실전 매매 전략
## 7.1 매수 시점 판단
### 7.1.1 역배열에서 골든크로스 발생
- 모든 이평선 하락 중 5일선이 10일·20일선 순차 돌파
- 추세 전환 초기 매수 기회
### 7.1.2 데드크로스이지만 매수해야 하는 경우
- 단기선(5일·20일) 데드크로스 but 장기선(120일·200일) 상승 추세 유지
- 상승 추세 속 일시적 조정 — 오히려 매수 기회
## 7.2 매도 시점 판단
### 7.2.1 정배열에서 데드크로스 발생
- 5일선이 10일·20일선 순차 하향 돌파
- 상승 추세 종료 신호
### 7.2.2 골든크로스이지만 매도해야 하는 경우
- 전체적으로 역배열인 상태에서의 단기 골든크로스
- 일시적 반등(Dead Cat Bounce)일 수 있음
## 7.3 이동평균선 활용 시 주의사항
- 이평선은 후행 지표 — 확인 신호이지 예측 도구가 아님
- 거래량, RSI 등 다른 지표와 함께 종합 판단 (MACD는 별도 글에서 다룰 예정)
- 종목의 특성(변동성, 거래량)에 따라 적합한 이평선 기간이 다름
- 과도한 기술적 분석 의존의 위험성

# 8. 마무리
```
