# 1. 개요

## 서버자동주문이란?

서버자동주문은 사용자가 미리 지정한 조건(가격, 수량 등)에 도달하는 경우 미래에셋증권의 서버가 자동으로 매수, 매도 주문을 실행해주는 기능이다. 직접 HTS/MTS를 켜놓지 않아도 (PC, 휴대폰이 꺼져도) 서버에서 실시간 감시, 주문이 가능하다는 것이 가장 큰 장점이다.

> 왜 사용해야 하는가?

서버자동주문은 직장이나 외출 등으로 시장을 실시간으로 지켜보기 어려운 투자자에게 특히 유용하다. 급격한 주가 변동에도 미리 설정한 손절(Stoploss)이나 이익 극대화 전략(Trailing Stop, 목표가 설정)을 통해 자동으로 대응할 수 있어 불필요한 손실을 막아준다. 또한 투자자의 심리를 배제한 ‘시스템적 매매’가 가능하다는 점에서 안정적인 트레이딩 환경을 제공하며, 한 번 설정하면 최대 90일까지 조건이 유지되기 때문에 단기뿐 아니라 중기적인 투자 전략에도 활용할 수 있다.

# 2. 서버자동주문에 대해서 알아보자

서버 자동 주문은 MTS나 HTS에서도 설정할 수 있다.

## 2.1 MTS (모바일 거래)

MTS에서 자동 매매 설정하는 방법은 다음과 같다.

1. 미래에셋 증권 앱 실행 > MY자산 에서 보유하고 있는 주식을 클릭하면 선택할 수 있는 버튼이 뜨고 자동 버튼을 클릭한다
2. 자동매도 감시조건 간단 설정으로 가게 되고 여기서 감시조건, 주문조건, 감시 유효시간 정도만 설정할 수 있다
   * 감시 유휴기간은 국내, 해외 주식 모두 최대 90일까지 걸어둘 수 있다

![image-20260223231122024](/Users/user/WebstormProjects/investment.advenoh.pe.kr/docs/start/auto-buy/image-20260223231122024.png)

![image-20260223231139300](/Users/user/WebstormProjects/investment.advenoh.pe.kr/docs/start/auto-buy/image-20260223231139300.png)

3. **저장후 실행** 버튼 클릭하고 **확인**을 누르면 자동 매도 설정이 완료된다

![image-20260223231206960](/Users/user/WebstormProjects/investment.advenoh.pe.kr/docs/start/auto-buy/image-20260223231206960.png)

1. ‘자동매수’ 또는 ‘자동매도’ 등록 및 감시 시작
2. 등록된 자동주문은 앱 내 ‘자동주문현황’에서 언제든 확인·수정 가능

* 예약주문과 다르게, 서버에서 항상 실시간 감시되어 스마트폰 종료 시에도 주문이 유효함

## 2.2 HTS (카이로스 PC 트레이딩)

* 기본 화면에 대한 설명
* 자동 매수/매도 설정

![image-20260223231226845](/Users/user/WebstormProjects/investment.advenoh.pe.kr/docs/start/auto-buy/image-20260223231226845.png)

* 주식 서버자동주문 화면 또는 ‘주식특화주문 > 서버자동주문’ 메뉴에서 사용
* 계좌/종목/수량/감시조건(Stoploss, 목표가, Trailing Stop) 등을 구체적으로 설정
* 감시 시작 후 서버에서 실시간 감시→조건 충족 시 자동 주문
* 이미 실행된 조건은 해제 불가, 감시 중인 조건은 언제든 중단·수정 가능
* Trailing Stop: 이익실현에 도달하면 자동으로 추가 이익 확정선(트레일 기준)을 따라가면서, 하락 전 고점 기준 일정폭 하락 시 자동 매도

### **Trailing Stop 등 고급 전략**

* 트레일링스탑은 기준가 돌파 후 고점을 자동 추적, 일정 폭(비율/가격) 하락 시 매도
* 스탑로스 대비 좀 더 ‘이익 극대화’에 중점
* 익절·손절 조건을 복수로 동시 설정 가능

## 2.3 해외주식 서버자동주문

해외 주식도 동일하다. HTS, MTS 확면 보여주기

* 해외주식도 국내 주식과 동일하게 설정할 수 있다
* MTS, HTS의 기본 화면 추가후 설정하기
* 해외주식 역시 ‘서버자동주문 지원’
* MTS/HTS에서 국내주식과 거의 동일한 방법으로 설정
* 감시 기간은 약 30일, 원화/외화 주문 모두 감시 가능
* 해외주식 자동주문도 손절/익절/트레일링스탑 사용 가능
* 해외주식 자동주문 시 거래소 상황·환율 등 추가 유의사항 숙지 필요

# 3. 마무리

미래에셋증권 서버자동주문은 “조건만 설정하면 실시간 감시는 서버에서, 주문 실행도 자동”이라는 점이 핵심입니다. 단순 예약주문보다 훨씬 강력하며, 트레이딩 자동화는 물론, 변동성이 심한 장에서 투자자 본인의 심리적 부담을 줄여줄 수 있습니다. 국내/해외주식, MTS/HTS 모두 적극 활용해 안정적이고 효율적인 투자 전략을 세워보시길 추천합니다.
