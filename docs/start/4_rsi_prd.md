# RSI(상대강도지수) 스터디 — 블로그 포스트 작성 PRD

## 1. 개요

### 1.1 목적

RSI(Relative Strength Index, 상대강도지수)의 개념, 계산 원리, 실전 활용법을 투자자 관점에서 체계적으로 정리한다. "과매수/과매도"라는 단어만 아는 수준에서, 실제 매매 판단에 활용할 수 있는 수준까지 이해하는 것이 목표다.

### 1.2 배경

- RSI는 가장 널리 사용되는 기술적 지표 중 하나지만, 단순히 70 이상 = 매도, 30 이하 = 매수로만 아는 경우가 많음
- 다이버전스, 추세 확인, 다른 지표와의 조합 등 깊이 있는 활용법 정리 필요
- 기술적 분석 시리즈의 첫 글로 적합

### 1.3 시리즈

- **시리즈명:** "차트 분석 가이드" (신규)
- 첫 번째 글: RSI

## 4. 콘텐츠 목차 구조

### 4.1 최종 목차 (안)

#### 1. 개요
- RSI란 무엇인가
- 왜 RSI를 알아야 하는가
- J. Welles Wilder가 1978년 개발 (《New Concepts in Technical Trading Systems》)

#### 2. RSI 계산 원리
- 공식: RSI = 100 - (100 / (1 + RS))
- RS(Relative Strength) = 평균 상승폭 / 평균 하락폭
- 기본 기간: 14일 (Wilder 권장)
- 단계별 계산 예시 (실제 숫자로)
- 왜 14일인가? 다른 기간(9일, 25일) 사용 시 차이

#### 3. RSI 읽는 법 — 기본
- RSI 범위: 0~100
- **70 이상**: 과매수(Overbought) — 가격이 단기 과열, 조정 가능성
- **30 이하**: 과매도(Oversold) — 가격이 과도하게 하락, 반등 가능성
- **50 기준선**: 상승 추세 vs 하락 추세 판단
- 주의: 과매수 = 무조건 매도가 아니다 (강한 상승장에서는 70 이상 유지 가능)

#### 4. RSI 다이버전스
- **상승 다이버전스(Bullish Divergence)**
  - 주가는 저점을 낮추는데 RSI는 저점을 높이는 경우
  - 하락 모멘텀 약화 → 반등 신호
- **하락 다이버전스(Bearish Divergence)**
  - 주가는 고점을 높이는데 RSI는 고점을 낮추는 경우
  - 상승 모멘텀 약화 → 조정 신호
- 다이버전스가 실패하는 경우 (강한 추세에서는 무시됨)
- 차트 예시

#### 5. RSI 실전 활용 전략
- **추세 확인 도구로 사용**: 50 위 = 상승 추세, 50 아래 = 하락 추세
- **Failure Swing**
  - Bullish Failure Swing: RSI가 30 아래 → 반등 → 다시 하락하지만 30 안 깨고 → 이전 고점 돌파
  - Bearish Failure Swing: 반대
- **RSI + 이동평균 조합**: RSI 신호 + 20일/60일 이동평균선 확인
- **RSI + 거래량**: 과매수/과매도에서 거래량 동반 여부로 신뢰도 판단
- **기간 설정 가이드**: 단기(9일) vs 기본(14일) vs 장기(25일)

#### 6. RSI의 한계와 주의사항
- 강한 추세장에서는 과매수/과매도 신호가 오래 지속됨 (속임 신호)
- RSI 단독 사용은 위험 — 다른 지표와 병행 필수
- 횡보장에서 가장 효과적, 강한 추세장에서는 효과 감소
- 종목/자산별로 과매수/과매도 기준 조정 필요 (80/20 등)

#### 7. 다른 보조지표와 비교
| 지표 | 특성 | RSI와의 차이 |
|------|------|-------------|
| MACD | 추세 추종형 | RSI는 모멘텀/반전, MACD는 추세 방향 |
| 스토캐스틱 | 모멘텀 오실레이터 | RSI보다 민감, 단기 신호 많음 |
| 볼린저 밴드 | 변동성 기반 | RSI와 함께 쓰면 과매수/과매도 신뢰도 향상 |
