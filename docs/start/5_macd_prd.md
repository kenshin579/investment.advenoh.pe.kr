# MACD 지표 완벽 가이드 — 블로그 포스트 작성 PRD

## 1. 개요

### 1.1 목적

MACD(Moving Average Convergence Divergence)의 개념과 원리를 정리하고, 이동평균선과의 관계 및 실전 매매 신호 해석 방법을 설명한다.

### 1.2 배경

- MACD는 이동평균선에서 파생된 대표적인 추세 추종 지표
- 이동평균선 글에서 MACD를 별도 글로 분리하기로 결정
- 이동평균선 → MACD → (추후 다른 보조지표) 순서로 기술적 분석 시리즈 구성 가능

### 1.3 선행 글

- 이동평균선 완벽 가이드 (`moving-average-guide`) — MACD의 기반 개념

## 4. 콘텐츠 목차 구조 (안)

```markdown
# 1. 개요
- MACD란 무엇인가
- 이동평균선과의 관계
- 이 글에서 다루는 범위

# 2. MACD의 기본 개념
## 2.1 MACD의 정의와 역사
- 제럴드 아펠(Gerald Appel)이 1970년대 개발
- 두 이동평균선의 수렴(Convergence)과 발산(Divergence)을 측정
## 2.2 MACD 계산 방법
- MACD 라인 = 12일 EMA - 26일 EMA
- 시그널 라인 = MACD의 9일 EMA
- 히스토그램 = MACD 라인 - 시그널 라인
## 2.3 MACD 차트 읽는 법
- MACD 라인, 시그널 라인, 히스토그램의 의미

# 3. MACD 매매 신호
## 3.1 MACD 골든크로스 / 데드크로스
- MACD 라인이 시그널 라인을 상향/하향 돌파
## 3.2 제로라인 돌파
- MACD가 0을 상향/하향 돌파하는 의미
## 3.3 히스토그램 분석
- 히스토그램의 증가/감소로 추세 강도 판단

# 4. 다이버전스
## 4.1 상승 다이버전스 (Bullish Divergence)
- 주가는 저점 갱신, MACD는 저점 상승 — 반등 신호
## 4.2 하락 다이버전스 (Bearish Divergence)
- 주가는 고점 갱신, MACD는 고점 하락 — 하락 전환 신호
## 4.3 다이버전스 활용 시 주의사항

# 5. MACD 활용 시 주의사항
- 후행 지표의 한계
- 횡보장에서의 속임 신호
- 다른 지표(거래량, RSI 등)와 병행 활용
```
