# 이격도 완벽 가이드 — 블로그 포스트 작성 PRD

## 1. 개요

### 1.1 목적

이격도(Disparity Index)의 개념과 계산 방법을 정리하고, 이동평균선과 현재 주가의 괴리를 활용한 매매 전략을 설명한다.

### 1.2 배경

- 이격도는 주가가 이동평균선으로부터 얼마나 떨어져 있는지를 백분율로 나타낸 지표
- 이동평균선 글의 그랜빌 8법칙 중 4법칙(과매도)과 8법칙(과매수)에서 이격도 개념이 등장하지만, 상세 설명은 이 글로 분리하기로 결정
- 이동평균선 → 이격도 순서로 자연스러운 학습 흐름 구성

### 1.3 선행 글

- 이동평균선 완벽 가이드 (`moving-average-guide`) — 이격도의 기반 개념

## 4. 콘텐츠 목차 구조 (안)

```markdown
# 1. 개요
- 이격도란 무엇인가
- 이동평균선과의 관계
- 이 글에서 다루는 범위

# 2. 이격도의 기본 개념
## 2.1 이격도의 정의
- 현재 주가와 이동평균선 사이의 괴리를 백분율로 표현
## 2.2 이격도 계산 방법
- 이격도(%) = (현재 주가 / N일 이동평균) x 100
- 100% 기준: 주가 = 이평선 (괴리 없음)
- 100% 초과: 주가가 이평선 위 (과매수 가능성)
- 100% 미만: 주가가 이평선 아래 (과매도 가능성)

# 3. 이격도 활용 기준
## 3.1 이동평균선 기간별 이격도 기준
- 20일 이격도: 과매수/과매도 기준값
- 60일 이격도: 과매수/과매도 기준값
- 120일 이격도: 과매수/과매도 기준값
## 3.2 종목 특성별 이격도 해석
- 대형주 vs 중소형주의 이격도 범위 차이
- 변동성이 큰 종목은 이격도 기준을 넓게 설정

# 4. 이격도를 활용한 매매 전략
## 4.1 과매도 구간에서 매수
- 이격도가 기준 이하로 하락 시 기술적 반등 기대
- 그랜빌 4법칙과의 연관
## 4.2 과매수 구간에서 매도
- 이격도가 기준 이상으로 상승 시 차익 실현 고려
- 그랜빌 8법칙과의 연관
## 4.3 이격도 활용 시 주의사항
- 강한 추세에서는 이격도가 오래 유지될 수 있음
- 단독 사용보다 다른 지표와 함께 종합 판단
```
