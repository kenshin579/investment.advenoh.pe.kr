# 1. 개요

주식 차트를 볼 때 이동평균선만으로는 추세의 강도나 전환 시점을 정확하게 포착하기 어렵다. 이동평균선이 교차하는 시점을 기다리다 보면 이미 상당 부분 움직임이 진행된 후인 경우가 많다. 이런 한계를 보완하기 위해 등장한 것이 바로 `MACD(Moving Average Convergence Divergence)`이다.

MACD는 두 이동평균선의 관계를 수치화한 지표로, 이동평균선이 서로 가까워지고(수렴) 멀어지는(발산) 정도를 측정한다. 이동평균선 차트에서 눈으로만 판단하던 것을 숫자와 그래프로 명확하게 보여주기 때문에, 추세의 방향과 강도, 전환 시점을 보다 객관적으로 파악할 수 있다.

이 글에서는 MACD의 기본 개념과 계산 방법부터, 매매 신호 해석, 다이버전스 활용, 그리고 실전에서의 주의사항까지 체계적으로 정리한다.

> 참고: 이 글은 [이동평균선 완벽 가이드](/stock/moving-average-guide)의 후속 글이다. 이동평균선의 기본 개념(SMA, 골든크로스/데드크로스 등)을 이해하고 있다면 더 쉽게 따라올 수 있다.

# 2. MACD의 기본 개념

## 2.1 MACD의 정의와 역사

MACD는 1979년 미국의 투자 분석가 제럴드 아펠(Gerald Appel)이 개발한 기술적 분석 지표이다. 이름 그대로 이동평균선의 수렴(Convergence)과 발산(Divergence)을 측정하는 도구이다.

수렴은 두 이동평균선이 서로 가까워지는 것을 의미하고, 발산은 서로 멀어지는 것을 의미한다. 단기 이동평균선과 장기 이동평균선의 간격이 변하는 양상을 통해 현재 추세의 방향, 강도, 전환 가능성을 파악할 수 있다.

MACD가 널리 사용되는 이유는 하나의 지표 안에 추세 추종(Trend Following)과 모멘텀(Momentum) 분석 기능을 동시에 갖추고 있기 때문이다. 추세가 지속될지 반전될지를 판단하는 데 있어 이동평균선 단독 분석보다 한발 앞선 신호를 제공한다.

## 2.2 EMA(지수이동평균)란?

MACD를 이해하려면 먼저 EMA(Exponential Moving Average, 지수이동평균)를 알아야 한다. 이동평균선 글에서 다룬 이동평균은 `SMA(Simple Moving Average, 단순이동평균)`이다. SMA는 기간 내 모든 종가에 동일한 가중치를 부여하지만, EMA는 최근 데이터에 더 높은 가중치를 부여한다.

### SMA와 EMA의 차이

| 구분 | SMA (단순이동평균) | EMA (지수이동평균) |
|------|-------------------|-------------------|
| 가중치 | 모든 데이터에 동일한 가중치 | 최근 데이터에 더 높은 가중치 |
| 반응 속도 | 느림 | 빠름 |
| 노이즈 민감도 | 낮음 (안정적) | 높음 (민감) |
| 주 용도 | 장기 추세 파악, 지지/저항 | 단기 추세 변화 감지, MACD 등 보조지표 |

### EMA 계산 방식

EMA는 전날의 EMA 값에 오늘의 종가를 일정 비율(가중치)로 반영하여 계산한다.

```
EMA = 전일 EMA + 평활 계수 * (당일 종가 - 전일 EMA)
```

여기서 `평활 계수(Smoothing Factor)`는 다음과 같이 구한다.

```
평활 계수 = 2 / (N + 1)
```

예를 들어 12일 EMA의 평활 계수는 `2 / (12 + 1) = 0.1538`이다. 이는 당일 종가가 약 15.4%의 비중을 차지한다는 뜻이다. 기간이 짧을수록 평활 계수가 커져서 최근 가격 변화에 더 민감하게 반응한다.

### MACD에서 EMA를 사용하는 이유

MACD의 핵심 목적은 추세의 변화를 빠르게 포착하는 것이다. SMA는 모든 데이터를 동등하게 취급하기 때문에 오래된 데이터의 영향으로 반응이 느리다. 반면 EMA는 최근 가격 변화에 더 민감하게 반응하므로, 추세 전환을 SMA보다 빠르게 감지할 수 있다.

예를 들어 주가가 갑자기 급등하면, SMA는 과거 낮은 가격들의 영향으로 천천히 올라가지만, EMA는 급등한 최근 가격에 높은 가중치를 주기 때문에 더 빠르게 반응한다. MACD가 이동평균선의 교차보다 먼저 추세 전환 신호를 줄 수 있는 것도 이 때문이다.

## 2.3 MACD 계산 방법

MACD는 세 가지 요소로 구성된다: MACD 라인, 시그널 라인, 히스토그램.

### MACD 라인

```
MACD 라인 = 12일 EMA - 26일 EMA
```

단기 EMA(12일)에서 장기 EMA(26일)를 뺀 값이다. 두 이동평균선의 간격을 수치로 나타낸 것이다.

- MACD 라인 > 0: 단기 EMA가 장기 EMA 위에 있다 (상승 추세)
- MACD 라인 < 0: 단기 EMA가 장기 EMA 아래에 있다 (하락 추세)
- MACD 라인이 커지고 있다: 두 EMA의 간격이 벌어지고 있다 (추세가 강해지고 있다)
- MACD 라인이 작아지고 있다: 두 EMA의 간격이 좁아지고 있다 (추세가 약해지고 있다)

### 시그널 라인

```
시그널 라인 = MACD 라인의 9일 EMA
```

MACD 라인 자체를 다시 9일 EMA로 평활화한 것이다. MACD 라인의 추세를 보여주며, MACD 라인과의 교차가 핵심 매매 신호가 된다.

### 히스토그램

```
히스토그램 = MACD 라인 - 시그널 라인
```

MACD 라인과 시그널 라인의 차이를 막대그래프로 표현한 것이다. 양수이면 0선 위에, 음수이면 0선 아래에 표시된다.

### 기본 설정값

MACD의 기본 설정은 `(12, 26, 9)`이다. 대부분의 차트 플랫폼에서 이 값이 기본으로 적용된다.

| 파라미터 | 기본값 | 의미 |
|---------|--------|------|
| 단기 EMA | 12일 | 약 2주간의 가격 추세 |
| 장기 EMA | 26일 | 약 1개월간의 가격 추세 |
| 시그널 EMA | 9일 | MACD 라인의 약 2주간 추세 |

> 참고: 12, 26, 9라는 숫자는 제럴드 아펠이 처음 제안한 값으로, 주 6일 거래 기준에서 2주(12일), 1개월(26일), 1.5주(9일)를 의미했다. 현재는 주 5일 거래지만 이 설정이 여전히 표준으로 사용된다.

## 2.4 MACD 차트 읽는 법

MACD 차트는 일반적으로 주가 차트 아래에 별도 영역으로 표시된다. 차트에서 확인해야 할 요소는 다음과 같다.

| 요소 | 표시 방식 | 의미 |
|------|----------|------|
| MACD 라인 | 실선 (보통 파란색) | 12일 EMA와 26일 EMA의 차이 |
| 시그널 라인 | 점선 또는 실선 (보통 빨간색) | MACD 라인의 9일 EMA |
| 히스토그램 | 막대그래프 | MACD 라인과 시그널 라인의 차이 |
| 제로라인 | 수평선 (0) | 12일 EMA와 26일 EMA가 같은 지점 |

MACD 차트를 처음 볼 때는 다음 순서로 읽으면 도움이 된다.

1. **제로라인 기준으로 위치 확인**: MACD 라인이 0 위에 있으면 상승 추세, 0 아래면 하락 추세
2. **MACD 라인과 시그널 라인의 교차 확인**: 교차 방향에 따라 매매 신호 판단
3. **히스토그램의 크기와 방향 확인**: 막대가 커지면 추세 강화, 작아지면 추세 약화

### 신호의 신뢰도

| 제로라인 위에서 데드크로스 | 매도 | 높음 | 상승 추세에서 하락 전환 |
| 제로라인 아래에서 데드크로스 | 매도 | 보통 | 이미 하락 중, 추가 하락 가능 |

## 3.2 제로라인 돌파

MACD 라인이 제로라인(0선)을 돌파하는 것도 중요한 신호이다. 제로라인은 12일 EMA와 26일 EMA가 정확히 같은 값인 지점이다.

### 제로라인 상향 돌파

MACD 라인이 0 아래에서 위로 돌파하면, 12일 EMA가 26일 EMA 위로 올라갔다는 뜻이다. 이것은 이동평균선 차트에서 단기선이 장기선을 상향 돌파하는 골든크로스와 동일한 의미이다.

반대로 MACD 라인이 0 위에서 아래로 내려가면, 12일 EMA가 26일 EMA 아래로 떨어졌다는 뜻이다. 이동평균선의 데드크로스와 같다.

## 3.3 히스토그램 분석

히스토그램은 MACD 라인과 시그널 라인의 차이를 시각적으로 보여준다. 교차 신호보다 한발 앞서 추세 변화의 조짐을 포착할 수 있다는 점에서 유용하다.

### 히스토그램으로 추세 강도 판단하기

- **양의 히스토그램이 커지는 경우**: 상승 모멘텀이 강해지고 있다
- **양의 히스토그램이 작아지는 경우**: 상승 모멘텀이 둔화되고 있다 (아직 하락은 아님)
- **음의 히스토그램이 커지는 경우**: 하락 모멘텀이 강해지고 있다
- **음의 히스토그램이 작아지는 경우**: 하락 모멘텀이 둔화되고 있다 (아직 상승은 아님)

<!-- TODO: TradingView에서 히스토그램 변화와 주가 흐름이 잘 보이는 차트 캡처 -->

핵심은 히스토그램의 "방향 변화"이다. 양의 히스토그램이 작아지기 시작하면 아직 MACD 라인이 시그널 라인 위에 있지만, 조만간 데드크로스가 발생할 수 있음을 미리 알려준다. 반대로 음의 히스토그램이 작아지기 시작하면 골든크로스가 임박했음을 시사한다.

```mermaid
graph LR
    A["히스토그램 양수 증가"] --> B["상승 추세 강화"]
    B --> C["히스토그램 양수 감소"]
    C --> D["상승 모멘텀 둔화"]
    D --> E["데드크로스 임박"]
    E --> F["히스토그램 음수 증가"]
    F --> G["하락 추세 강화"]
    G --> H["히스토그램 음수 감소"]
    H --> I["하락 모멘텀 둔화"]
    I --> A
```

# 4. 다이버전스

다이버전스(Divergence)는 MACD에서 가장 강력한 신호 중 하나이다. 주가의 움직임과 MACD의 움직임이 서로 다른 방향으로 갈 때 발생하며, 현재 추세가 약해지고 있어 곧 반전될 가능성이 있음을 경고한다.

## 4.1 상승 다이버전스 (Bullish Divergence)

주가는 이전 저점보다 더 낮은 저점을 기록하는데, MACD는 이전 저점보다 더 높은 저점을 기록하는 경우이다.

<!-- TODO: TradingView에서 상승 다이버전스 차트 캡처 (주가 저점 하락 + MACD 저점 상승) -->

```
주가:  저점 A ───────────── 저점 B (A보다 낮음)
MACD:  저점 A' ──────────── 저점 B' (A'보다 높음)
→ 상승 다이버전스: 하락 추세의 힘이 약해지고 있다
```

주가는 계속 내려가고 있지만, MACD가 보여주는 하락 모멘텀은 오히려 줄어들고 있다는 뜻이다. 매도 세력이 약해지고 있으며, 조만간 반등이 나올 가능성이 있다.

상승 다이버전스는 하락 추세의 바닥 근처에서 나타나는 경우가 많아, 매수 진입을 고려할 수 있는 신호이다. 다만 다이버전스가 발생했다고 해서 즉시 반등하는 것은 아니며, 실제 반등이 확인된 후에 진입하는 것이 안전하다.

## 4.2 하락 다이버전스 (Bearish Divergence)

주가는 이전 고점보다 더 높은 고점을 기록하는데, MACD는 이전 고점보다 더 낮은 고점을 기록하는 경우이다.

<!-- TODO: TradingView에서 하락 다이버전스 차트 캡처 (주가 고점 상승 + MACD 고점 하락) -->

```
주가:  고점 A ───────────── 고점 B (A보다 높음)
MACD:  고점 A' ──────────── 고점 B' (A'보다 낮음)
→ 하락 다이버전스: 상승 추세의 힘이 약해지고 있다
```

주가는 계속 올라가고 있지만, MACD가 보여주는 상승 모멘텀은 줄어들고 있다. 매수 세력이 약해지고 있으며, 조만간 하락 전환이 나올 수 있다.

하락 다이버전스는 상승 추세의 천장 근처에서 나타나는 경우가 많다. 보유 중인 주식이 있다면 매도 또는 비중 축소를 고려할 수 있는 신호이다.

## 4.3 다이버전스 활용 시 주의사항

다이버전스는 강력한 신호이지만, 몇 가지 주의할 점이 있다.

**다이버전스는 타이밍 지표가 아니다.** 다이버전스가 발생했다고 즉시 추세가 반전되는 것은 아니다. 다이버전스가 나타난 후에도 기존 추세가 한동안 지속되는 경우가 많다. 다이버전스는 "추세의 힘이 약해지고 있다"는 경고이지 "지금 당장 반전된다"는 신호가 아니다.

**강한 추세에서는 다이버전스가 무시될 수 있다.** 강한 상승장이나 하락장에서는 다이버전스가 여러 번 연속으로 발생해도 추세가 계속될 수 있다. 특히 강세장에서의 하락 다이버전스를 근거로 공매도하는 것은 위험하다.

**다른 신호와 함께 확인해야 한다.** 다이버전스 단독으로 매매 결정을 내리기보다는, MACD 골든크로스/데드크로스, 주가의 지지/저항선 돌파, 거래량 변화 등을 함께 확인해야 신뢰도가 높아진다.

# 5. MACD 활용 시 주의사항

MACD는 유용한 지표이지만, 한계를 이해하고 사용해야 한다.

**후행 지표라는 본질적 한계가 있다.** MACD 자체가 이동평균선에서 파생된 지표이므로 후행성을 가진다. EMA를 사용하여 SMA 기반보다는 빠르지만, 여전히 과거 데이터를 기반으로 계산된다. 추세가 이미 상당히 진행된 후에야 신호가 나타나는 경우가 많다.

**횡보장에서 속임 신호가 잦다.** MACD는 추세 추종 지표이기 때문에, 뚜렷한 추세가 없는 횡보장에서는 MACD 라인과 시그널 라인이 빈번하게 교차하면서 거짓 신호를 반복한다. 횡보장에서는 MACD보다 RSI(상대강도지수)나 스토캐스틱 같은 진동(oscillator) 지표가 더 유용하다.

<!-- TODO: TradingView에서 횡보장에서 MACD 속임 신호가 반복되는 차트 캡처 -->

**MACD 단독으로 매매 결정을 내리면 안 된다.** 어떤 기술적 지표든 단독으로 사용하면 신뢰도가 떨어진다. MACD는 다음 요소들과 함께 종합적으로 판단할 때 효과적이다.

| 보조 수단 | 활용 방법 |
|----------|----------|
| 거래량 | MACD 신호 발생 시 거래량 증가 여부로 신뢰도 판단 |
| 이동평균선 배열 | 정배열/역배열 상태에서 MACD 신호의 맥락 해석 |
| RSI | 과매수/과매도 구간에서 MACD 신호와의 일치 여부 확인 |
| 지지/저항선 | 주요 가격대에서의 MACD 신호가 더 의미 있음 |

**기본 설정(12, 26, 9)이 모든 상황에 최적은 아니다.** 변동성이 큰 종목이나 단기 매매에서는 더 짧은 기간(예: 8, 17, 9)을, 장기 투자에서는 더 긴 기간(예: 19, 39, 9)을 사용하는 것이 더 효과적일 수 있다. 다만 기본 설정을 먼저 충분히 이해한 후에 조정하는 것이 바람직하다.

# 6. 마무리

MACD는 이동평균선의 한계를 보완하면서도 비교적 직관적으로 사용할 수 있는 기술적 분석 지표이다. 이 글에서 다룬 핵심 내용을 정리하면 다음과 같다.

- **MACD의 본질**: 두 EMA의 차이를 수치화한 것으로, 추세의 방향과 강도를 동시에 보여준다
- **EMA를 사용하는 이유**: SMA보다 최근 가격 변화에 민감하게 반응하여, 추세 전환을 더 빠르게 포착한다
- **세 가지 구성 요소**: MACD 라인(추세 방향), 시그널 라인(매매 타이밍), 히스토그램(추세 강도 변화)
- **매매 신호**: MACD-시그널 교차는 진입/청산 타이밍, 제로라인 돌파는 추세 전환 확인에 활용한다
- **다이버전스**: 주가와 MACD의 방향이 다를 때 추세 반전 가능성을 경고하는 강력한 신호이다
- **한계를 인식해야 한다**: 후행성, 횡보장에서의 속임 신호를 감안하여 거래량, RSI 등 다른 지표와 함께 종합 판단해야 한다
